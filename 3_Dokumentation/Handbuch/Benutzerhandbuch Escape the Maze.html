Installationsanleitung

Schritt 1: Verknüpfung erstellen:
  • Rechtsklick auf Desktop -> "Neu" -> "Verknüpfung"




  • Speicherort des Elemtents: javaw Startmenue



  • "Weiter" -> Verknüpfung benennen (Escape the Maze) -> Verknüpfung erstellen

Schritt 2: Verknüpfung anpassen:
  • Rechtsklick auf Verknüpfung -> "Eigenschaften" -> Feld "Ausführen in:" durch Speicherpfad der Class im "bin"-Ordner ersetzen




      • Speicherort herausfinden:

      • Speicherort des Projekts öffnen -> "bin"-Ordner öffnen -> Adresspfad kopieren



  • Kopierten Adresspfad in "Ausführen in:" einfügen

  • Änderungen speichern und fertig ist die Verknüpfung


Fehlerursachen beheben:

Korrekte Schreibweise der Verknüpfung überprüfen bzw. noch einmal Schritt für Schritt eine neue Verknüpfung erstellen.
Auch den Speicherort des Projekts noch einmal prüfen und den Pfad erneut kopieren und in "Ausführen in: einfügen.

Desweiteren ist für die Erstellung eines Desktopicons ist das Javatool "javaw" wichtig.
Da dieses Bestandteil des Java Development Kits (JDK) ist sollte sichergestellt werden, dass dieses (richtig) installiert ist.

Hierzu:
  • JDK deinstallieren: Start -> Einstellungen -> Apps -> "Java(TM) SE Development Kit" -> Klick darauf -> Deinstallieren



  • JDK neu installieren: neuste JDK Version aus "https://www.oracle.com/java/technologies/downloads/#jdk18-windows" herunterladen und anschließend installieren.
