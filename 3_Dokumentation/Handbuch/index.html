Installationsanleitung

Schritt 1: Verknüpfung erstellen:
  • Rechtsklick auf Desktop -> "Neu" -> "Verknüpfung"


  • Speicherort des Elemtents: javaw Startmenue

  • "Weiter" -> Verknüpfung benennen (Escape the Maze) -> Verknüpfung erstellen

Schritt 2: Verknüpfung anpassen:
  • Rechtsklick auf Verknüpfung -> "Eigenschaften" -> Feld "Ausführen in:" durch Speicherpfad der Class im "bin"-Ordner ersetzen


    • Speicherort herausfinden:
      • Speicherort des Projekts öffnen -> "bin"-Ordner öffnen -> Adresspfad kopieren

    • Kopierten Adresspfad in "Ausführen in:" einfügen

    • Änderungen speichern und fertig ist die Verknüpfung
